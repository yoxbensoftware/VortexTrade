- Sayfalama desteği: 200'e kadar BTC ticker çekebilir (2 sayfa × 100)
- Yüksek spread uyarısı (>1% kırmızı renk)
- Versiyon V.0.0.14'e yükseltildi

### V.0.0.16 — 2026-04-13

| Alan | Değer |
|---|---|
| **Tarih** | 2026-04-13 |
| **Geliştiren** | GitHub Copilot (AI) |
| **Bilgisayar** | PCDELL |

**Yapılan Geliştirme:**
- SOLID mimarisine uygun gerçek zamanlı piyasa verisi altyapısı oluşturuldu
- **IMarketDataProvider** interface'i oluşturuldu (`Services/Interfaces/IMarketDataProvider.cs`) — Tek sorumluluk: veri çekme, erişilebilirlik kontrolü, provider adı
- **IMarketDataStream** interface'i oluşturuldu (`Services/Interfaces/IMarketDataStream.cs`) — Sürekli veri akışı: DataReceived, ErrorOccurred, ProviderChanged event'leri, Start/Stop yaşam döngüsü
- **MarketDataEventArgs** ve **MarketDataErrorEventArgs** event argümanları oluşturuldu
- **MarketTicker** universal model oluşturuldu (`Models/MarketTicker.cs`) — Exchange, BaseCurrency, QuoteCurrency, Price, PriceUsd, Volume, VolumeUsd, LastTraded, Source
- **BinanceMarketDataProvider** oluşturuldu (`Services/MarketData/BinanceMarketDataProvider.cs`) — Binance REST API `/api/v3/ticker/24hr`, BTC çiftlerini filtreler, sembol parse, ping ile erişilebilirlik kontrolü
- **CoinLoreMarketDataProvider** oluşturuldu (`Services/MarketData/CoinLoreMarketDataProvider.cs`) — CoinLore API, çoklu coin desteği (BTC/ETH/XRP/SOL ID map), önceki CoinLoreMarketDataService'in SOLID uyumlu yeniden yazımı
- **MarketDataStream** polling engine oluşturuldu (`Services/MarketData/MarketDataStream.cs`) — Öncelik sıralı provider fallback mekanizması (Binance→CoinLore), 3 ardışık hatada otomatik provider geçişi, SynchronizationContext ile UI thread güvenliği, observer pattern ile çoklu consumer desteği
- **BtcTickerForm** refactor edildi — Artık kendi timer/service yönetimi yerine IMarketDataStream event'lerine abone oluyor, aktif provider bilgisi ekranda gösteriliyor, provider değişikliği ve hata bildirimi event-based
- Mimari avantajlar: Farklı ekranlar aynı IMarketDataStream'e abone olabilir, analiz ve strateji motorları provider'ları bağımsız kullanabilir, yeni provider eklemek tek sınıf yazmak kadar kolay (Open/Closed Principle)
- Versiyon V.0.0.16'ya yükseltildi
